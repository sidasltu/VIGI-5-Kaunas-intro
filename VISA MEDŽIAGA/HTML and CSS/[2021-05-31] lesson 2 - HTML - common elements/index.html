Primary HTML elements

There are over 100 HTML elements, but most of them are out-dated or rarely used. There are about 20 primary/common elements which every web developer should know and use correctly.

What is HTML element?

HTML element is a special syntax, which purpose is to structure page and it's content. Most HTML elements have opening and closing tag. Opening tag is wrap in diamond quotes:
<HTML tag name>
Closing tag is writen similary, but with slash before tag name:
</HTML tag name>

Everything between opening and closing tag is interpreted as children components.

Attributes are additional setting for HTML element. They should be writen in element's opening tag. There are common attributes for all elements like: class, style, etc.. Some HTML elements have specific attributes, for instance:
<img> has 'alt' and 'src' attributes
<a> has 'href' and 'target' attributes
... etc.

Inline Elements

Inline elements are interpreted as text. These elements can't have spacing settings. This kind of elements are positioned from left to right, and only if there are no more space, they are placed in the next row. Inline elements can apply all text-style properties, such as

  • color - text color
  • text-align - horizontal alignment
  • vertical-align - vertical alignment
  • letter-spacing - spacing between letters
  • line-height - height of one line
  • ... etc.

All typography(text) style properties are applied to element and all of it's descending children!
